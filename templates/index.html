Comprehensive RESTful API for downloading and managing Telegram stories

Base URL
GET /api/current Fetch active stories List Stories

Retrieve all currently active (non-expired) stories from a Telegram user. Returns metadata for each story including ID, type, date, and caption.

Query Parameters

Parameter Type Required Description
username string Yes Telegram username without @ symbol

Example Request

cURL 
curl -X GET "/api/current?username=durov"

GET /api/special Download specific story Download

Download a specific story by providing username and story ID. Returns a temporary download URL valid for 60 minutes along with complete story metadata.

Query Parameters

Parameter Type Required Description
username string Yes Telegram username without @ symbol
storyid integer Yes Unique story identifier (numeric)

Example Request

cURL 
curl -X GET "/api/special?username=durov&storyid=1234"

Response Format

200 OK Success 
{
  "success": true,
  "username": "durov",

GET /api/direct Download via URL Direct Link

Download story directly using a Telegram story URL. Automatically parses username and story ID from the URL. Supports multiple URL formats.

Query Parameters

Parameter Type Required Description
url string Yes Full Telegram story URL (URL-encoded)

Supported URL Formats

https://t.me/username/s/storyid Standard story link format
https://telegram.me/username/s/storyid Alternative domain format
https://t.me/c/channelid/storyid Private channel story format

Example Request

cURL 
curl -X GET "/api/direct?url=https://t.me/durov/s/1234"

Response Format

200 OK Success 
{
  "success": true,
  "username": "durov",
  "story_id": 1234,
  "type": "Pinned",
  "media_type": "photo",
  "date": "2026-01-09 14:30:00",
  "timestamp": 1736432200,
  "caption": "Amazing photo!",
  "download_url": "https://tmpfiles.org/dl/789012/story.jpg",
  "expires_in": "60 minutes",
  "source_url": "https://t.me/durov/s/1234",
  "api_dev": "@ISmartCoder",
  "api_channel": "@abirxdhackz"
}
400 Bad Request 
{
  "success": false,
  "error": "Invalid Telegram story URL format. Expected: https://t.me/username/s/storyid",
  "api_dev": "@ISmartCoder",
  "api_channel": "@abirxdhackz"

Response Fields Reference

Common Fields

success Boolean indicating request status
error Error message (only on failure)
api_dev API developer contact
api_channel Official Telegram channel

Story Object

story_id Unique story identifier
type Active | Pinned | Archived
media_type photo | video | document
caption Story caption text (may be empty)

Download Fields

download_url Temporary file download link
expires_in Link expiration time (60 min)
date Human-readable date string
timestamp Unix timestamp (seconds)

Important Notes & Limits

Rate Limiting

API requests are subject to Telegram's rate limits. Excessive requests may result in temporary blocks. Recommended: max 30 requests/minute.

Download Link Expiry

Download URLs are temporary and expire after 60 minutes. Store content locally if you need it for longer periods.

No Authentication Required

All endpoints are publicly accessible without API keys. For private channels, ensure the service account has access.
